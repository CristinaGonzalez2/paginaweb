Funciones de excel:

¿Que es una función?

Las funciones de excel son fórmulas predefinidas que permiten realizar cálculos, análisis de datos y manipulación de información de manera eficiente. Cada función tiene un nombre y puede incluir argumentos, que son los datos necesarios para realizar el cálculo.
Algunas funciones comunes incluyen:

Suma: Suma de un rango de celdas
Promedio:

Calcula el promedio de un conjunto de valores
